# Utility for Point Blank Zepetto ID

## Features:
- Macro (3Q scope {use RightClick to trigger 3Q Scope. the trigger itself is open scope then automatically fire} & 3Q)
- Crosshair with various color

Currently work for Point Blank, and perhaps for mostly fps games

## Prerequisite
- Visual studio with C/C++ development kit

## How to use:
- clone the repo git clone https://github.com/ItsMe-RiiK/PBUtility.git
- open slnx file to go to the visual studio
- retarget all and apply
- change to Release x64 if needed
- build the solution
- Run the exe. the exe by default already always run as administrator

The application.exe would appear in Release folder

## Hotkeys:
- f1 to enable/disable macro
- f2 to enable/disable crosshair
- f3 to change macro mode (3Q Scope or just 3Q)
- f4 to change crosshair mode (dot and plus)
- f5 to change crosshair color
